Agent disagreement as a direction vector into unexplored search space

The insight: Two agents solving the same problem produce two compositions on S³. Their agreement zones are confirmed constraints — the solution must pass through here. Their disagreement zones are direction vectors — the gap quaternion points at unexplored space.

W-channel divergence (■ amber): one agent believes an action exists in the solution, the other doesn't. Different world models at that point. RGB-channel divergence (■ purple): same actions, different sequence. Same world model, different planning heuristics.

A third agent that starts at the frontier — constrained by coherent zones, guided by the gap vector — searches exactly the space neither agent has explored. Not randomly. Algebraically guided.

Connection to Closure Ea

Closure Ea enables the dynamic version: a third Enkidu cell whose adapter truth is the gap lattice between Agent A and Agent B. As each agent produces new events, the gap evolves. The third cell's closure condition is: reduce the gap sigma to zero. It lives through the disagreement, learns what resolves it, and emits the synthesis upward when the gap closes.

The polarity engine is Step 6 in the universal staircase: "Mixed world signals — action + state + consequence streams." Two streams whose closure structures have been mapped. The disagreement IS the adapter truth for the synthesis cell. One lattice describing another.
